Infraestructura Windows Server – TechSolutions S.L.

Transformación integral de la infraestructura tecnológica para una consultora con 60 empleados. El proyecto consistió en migrar de un modelo de "Grupo de Trabajo" descentralizado e inseguro a un dominio centralizado gestionado con Active Directory.

Se implementó una estructura jerárquica de Unidades Organizativas (OU), se automatizó la creación de usuarios mediante scripts de PowerShell y se desplegaron servicios críticos de red (DNS y DHCP) para garantizar la conectividad y la resolución de nombres interna. Además, se establecieron políticas de seguridad estrictas mediante GPO y un sistema de almacenamiento redundante.

Detalles del proyecto

La solución abordó la falta de control de identidades y la vulnerabilidad de los datos. Se configuró un servidor de archivos con permisos NTFS y cuotas de almacenamiento, asegurando que cada departamento tuviera acceso únicamente a su información pertinente, eliminando el riesgo de fugas de datos internas.

Cliente TechSolutions S.L. Rol Administrador de Sistemas Año 2025 Tecnologías Windows Server 2022, Active Directory (AD DS), PowerShell, DNS, DHCP, RAID 1/5 Infraestructura Servidor Dell PowerEdge (Simulado) Recuperación Backups diarios programados (22:00h)

Resultados alcanzados

  • Centralización total de 60 identidades de usuario y equipos
  • Automatización del despliegue de usuarios (reducción de horas a minutos)
  • Alta disponibilidad mediante RAID 1 (SO) y RAID 5 (Datos)
  • Estandarización del entorno visual y seguridad mediante GPO
